What is DevSecOps?

DevSecOps is the addition of security to DevOps. It is an overall process to ensure that security is “baked in” to the entire software development cycle.

DevSecOps is a logical extension of DevOps:
1.DevOps integrates operations into the develop/release cycle. DevSecOps integrates security into the develop/release cycle.

2.DevOps increases the speed at which software is developed and delivered. DevSecOps increases the security with which software is developed and delivered.

3.DevOps automates much of the software lifecycle. DevSecOps requires merging and automating many of the traditional practices of security engineers, operations teams, and development teams.
